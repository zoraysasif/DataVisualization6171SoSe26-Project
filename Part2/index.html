Brush axes to filter candidates globally.

Why these axes? We have optimized these dimensions specifically for Automotive Engine Block Design. We start with critical Mixed Composition elements (Si, Cu, Mg, Fe) that drive the alloy's behavior. Manufacturability (CSC, delta_T) ensures the block can be cast without hot tearing. Microstructure (Vf_DIAMOND_A4) tracks the primary silicon phase essential for cylinder wear resistance. Finally, we evaluate the resulting Mechanical (Yield Strength, Hardness) and Thermo-Physical (Conductivity, Thermal Expansion) properties to guarantee the block withstands combustion pressures and thermal gradients.
